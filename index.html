Notion API

Where the Publish to Notion button POSTs the markdown. Deploy api/publish.js to Vercel (see README), then paste the URL here.

Step-by-step onboarding (Vercel + Notion integration)
  1. Create a Notion internal integration at notion.so/my-integrations. Copy the Internal Integration Secret — that's your NOTION_TOKEN.
  2. In Notion, open the parent page (or database) where you want plans to live. Click ··· → Add connections → pick the integration you just made. Copy the page/database ID from the URL.
  3. Sign in to vercel.com with GitHub → Add New → Project → import josiah-lgtm/reverse-engineering. Framework preset Other, output directory .
  4. In Vercel Project Settings → Environment Variables, add: NOTION_TOKEN and NOTION_PARENT_DATABASE_ID (or NOTION_PARENT_PAGE_ID).
  5. Deploy. Copy the assigned URL like reverse-engineering-xyz.vercel.app.
  6. Paste https://reverse-engineering-xyz.vercel.app/api/publish into the Publish endpoint field above.
